# Design summary

- Design concept: Programmer sprint board
- Layout personality: technical agile
- Typography: IBM Plex Mono headings with IBM Plex Sans body text
- Color palette: primary #0f172a, secondary #34d399, accent #60a5fa, background #f8fafc, text #0f172a, muted #64748b
- Shape language: Sharp corners
- Visual motif: sprint lanes and code tabs
- Unique layout choices: first sections use Sprint board, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
